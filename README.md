Employee Engagement & Burnout Diagnostic Analysis

About the Project :
In a high-performance organization like Palo Alto Networks, reactive HR interventions are often too late. This project shifts HR analytics from a reactive model to a preventive diagnostic system. By analyzing engagement signals, burnout risks, and career satisfaction, this dashboard provides HR leaders with early-warning indicators to mitigate attrition and improve employee well-being.

Key Features :
Engagement Index: A composite score derived from Job Involvement, Job Satisfaction, Environment Satisfaction, and Relationship Satisfaction.

Burnout Risk Detection: Uses overtime intensity and work-life balance metrics to flag high-risk employee segments.

Interactive Dashboard: A Streamlit-based web app allowing HR managers to filter by department and identify "Priority Intervention" groups.

Preventive Insights: Identifies attrition drivers like career stagnation and departmental burnout hotspots.

Analytical Insights :
Our analysis of the dataset revealed critical findings:

Burnout-Attrition Link: Employees classified as "High Burnout Risk" have a 33% attrition rate compared to 9% in the low-risk group.

Departmental Hotspots: R&D department exhibits the highest average burnout risk scores.

Career Stagnation: Engagement scores show a sharp decline for employees with over 15 years in the same role.

Tech Stack :
Language: Python 3.x

Data Analysis: Pandas, NumPy

Visualization: Seaborn, Matplotlib

Web Framework: Streamlit

Environment: Windows/Linux
